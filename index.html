HOW TO USE - PGA League System

Overview

The PGA League System is a comprehensive golf management application with 5 main tabs: League, Handicap, Scorecard, Winners, and SWINDLE Prizes.

1. LEAGUE TAB

Purpose:

Track player standings across multiple weeks of competition.

Key Features:

  • Add Players: Click "Add Player" to add new players to the league
  • Enter Scores: Click on any week cell to enter a player's score for that week
  • Best Scores: Use the "Best X" dropdown to select how many top scores count toward total
  • Adjust Weeks: Click "Adjust Weeks" to add or remove weeks from the season
  • Export Data: Click "Export CSV" to download league standings
üí° Tip: Top scores for each player are highlighted in gold. The system automatically calculates totals based on your "Best X" selection.

2. HANDICAP TAB

Purpose:

Import player data from Excel files and manage handicap information.

Steps to Use:

  1. Upload Excel File: Click "Choose File" and select your Excel handicap sheet
  2. Extract Data: Click "Extract Data" to import player information
  3. Set Payments: Click on "Paid" cells to toggle payment status (changes between "5" and "0")
  4. Create Draw: Click "Create" under Draw Management to organize players into groups
  5. Capture Images: Use "Capture" to save screenshots of the draw or payment list
üí° Tip: Green values indicate paid status, red indicates unpaid. Use "All ¬£5" or "All ¬£0" buttons to quickly set all payments.

3. SCORECARD TAB

Purpose:

Record detailed hole-by-hole scores and calculate Stableford points.

Steps to Use:

  1. Select Week: Choose which week you're recording scores for (WK1, WK2, etc.)
  2. Course Info: Enter course name, date, and competition details
  3. Add Players: Click "Add Player" and enter player name and handicap
  4. Enter Scores: Input scores for each hole (1-18)
  5. View Stableford: The system automatically calculates Stableford points below each score
  6. Backup Data: Click "Backup Scorecard" to save a JSON file
  7. Capture Image: Click "Capture Scorecard" to save as image
‚ö†Ô∏è Important: Each week's scorecard is stored separately. Make sure you've selected the correct week before entering scores!

4. WINNERS TAB

Purpose:

Manage winners, prizes, and view handicap calculations.

H'CAP Calculator Section:

  • Select Week: Choose which week to view handicap calculations
  • Show All Players: Toggle to see all players or only those with scores
  • Refresh Data: Sync latest data from Handicap tab
  • Generate Report: Create a downloadable .txt report for the current week

Winners - Saturday Swindle Section:

  1. Select Week: Choose the week you want to manage winners for
  2. Import Scorecard Players: Click to automatically import players from the Scorecard tab (sorted by Stableford score)
  3. Auto-Prizes: The system automatically populates prize amounts based on player count and the prize matrix
  4. Edit Prizes: You can manually edit any prize amount if needed
  5. Clear Week: Remove all winners data for the selected week
  6. Clear ALL Weeks: Delete all winners data (use with caution!)

Profit & Loss Section:

Click "Profit & Loss" button to view comprehensive financial analysis:

  • Shows each player's total weeks played
  • Calculates total loss (¬£5 per week played)
  • Shows total prizes won
  • Displays net profit/loss (green = profit, red = loss)
  • Download as CSV for external analysis

Golf Handicap Cut & Increase Calculator:

Individual calculator for determining handicap adjustments based on:

  • Player's current index and category
  • Prize money won
  • Stableford points scored
  • Finishing position in swindle
  • Tee color (White/Yellow)
üí° Tip: The system automatically divides Stage 1 cuts by finishing position (1st gets full cut, 2nd gets half, etc.).

5. SWINDLE PRIZES TAB

Purpose:

View the prize distribution matrix for different player counts.

How It Works:

  • Shows prize breakdown for 2-50 players
  • Each player contributes ¬£5 to the pot
  • Prizes distributed among top finishers (1st, 2nd, 3rd, 4th, 5th depending on player count)
  • This matrix is used automatically when importing players to Winners tab

Example: With 17 players:

  • Total pot = ¬£85
  • 1st place = ¬£43
  • 2nd place = ¬£26
  • 3rd place = ¬£16

GENERAL FEATURES

Data Management:

  • Backup: Click the "Backup" button in the header to download all data as JSON
  • Restore: Click "Restore" to upload a previously saved backup file
  • Theme: Click "Theme" to choose from 10 different color schemes
‚ö†Ô∏è Data Storage: All data is stored in your browser's local storage. Regular backups are recommended to prevent data loss!

WORKFLOW RECOMMENDATIONS

Weekly Process:

  1. Before the Round:
    • Go to Handicap tab
    • Upload latest handicap Excel file
    • Extract data and verify all players
    • Create draw for the day
    • Capture and share draw image
  2. During/After the Round:
    • Go to Scorecard tab
    • Select current week
    • Add all players and enter their scores
    • Verify Stableford totals
    • Backup scorecard data
  3. After the Round:
    • Go to Winners tab
    • Import players from Scorecard (auto-sorted by score)
    • Verify prize amounts (auto-populated)
    • Adjust prizes if needed
    • View Profit & Loss for season overview
  4. League Update:
    • Go to League tab
    • Enter top Stableford scores for each player
    • Update standings
    • Export CSV if needed for sharing
